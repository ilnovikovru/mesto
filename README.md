# Проект: Место
------
Лендинг о местах, которые можно посетить в России: https://ilnovikovru.github.io/mesto/

## Функциональность
Экран с текстом, карточками, иллюстрациями. Можно изменить имя и профессию. Работает на любых устройствах.

## Технологии
1. HTML5
2. CSS
3. Флексбоксы
4. Адаптивная верстка
5. БЭМ
6. JavaScript

## Планы по доработке проекта
Исправить код по замечаниям ревьюера с помощью материалов курса «Веб-разработчика» и справочной информации в интернете, чтобы получить готовый проект.
